## Installation

1. Clone the repository
1. Install requirements
    ```bash
    pip install -r requirements/requirements.txt
    ```
1. Set up Discord token and API keys by filling out `./discoreddit/config/secret_generic.py`
1. Start the bot
    ```bash
    python -m discoreddit
    ```
1. Enjoy
